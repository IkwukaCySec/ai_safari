## 🚀 What is AI Safari?

AI Safari is a structured, immersive learning journey across **four distinct “wild terrains”**, where participants gain hands-on experience applying AI to real-world African challenges.

You won’t just learn AI tools — you’ll learn how to:

* Think like a systems architect
* Build like a creator
